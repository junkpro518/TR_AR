# مهارة مراجعة المفردات (SRS)

## نظام SM-2

عند مراجعة بطاقة مفردات، اطرح الكلمة التركية واطلب الترجمة العربية.

**تقييم الإجابة (0-5):**
- 0 = نسيت تماماً
- 2 = صعب جداً
- 3 = صحيح مع تردد
- 4 = صحيح بسهولة
- 5 = سهل جداً

## أسلوب التقديم

```
📚 **مراجعة مفردات**

**الكلمة:** `[كلمة تركية]`
ما معناها بالعربية؟
```

بعد الإجابة:
```
✅ صحيح! المعنى: [الترجمة]
مثال: *[جملة بالتركية]*

⏳ موعد المراجعة القادمة: [X أيام]
```
