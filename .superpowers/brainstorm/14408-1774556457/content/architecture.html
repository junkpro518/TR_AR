Next.js + Supabase + Claude API

طبقات التطبيق

🖥️ Frontend — Next.js (App Router)
صفحة المحادثة /chat لوحة التقدم /dashboard إعدادات الأهداف /goals سجل الجلسات /history
↕ API Routes (Next.js Server)
⚙️ API Layer — Next.js API Routes
POST /api/chat — إرسال رسالة لـ Claude POST /api/feedback — تحليل لغوي GET /api/session — جلب/حفظ جلسة
🤖 Claude API
نموذجان منفصلان:
claude-sonnet-4-6 للمحادثة claude-haiku-4-5 للتحليل السريع
🗄️ Supabase
Auth (بدون تسجيل — مستخدم واحد) sessions — جلسات المحادثة messages — الرسائل goals — أهداف التعلم feedback_log — سجل الأخطاء vocab_progress — نمو المفردات
🚀 النشر
Vercel — Frontend + API Supabase Cloud — DB

تدفق الرسالة الواحدة

المستخدم يكتب → POST /api/chat → Claude (محادثة) → رد المعلم
في نفس الوقت → POST /api/feedback → Claude Haiku (تحليل) → تغذية راجعة فورية
بعد الرد → Supabase → حفظ الرسالة + الأخطاء + تحديث الإحصائيات
