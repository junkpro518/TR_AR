Next.js + Supabase + OpenRouter مع أنظمة التعلم الأربعة

🏗️ المعمارية المحدّثة

Frontend (Next.js App Router) → API Routes → OpenRouter API (أي نموذج) + Supabase

🧠 أنظمة التعلم الأربعة

1. SRS — تكرار متباعد
• كل مفردة جديدة تُضاف لبطاقة SRS
• خوارزمية SM-2 تحدد موعد المراجعة
• قسم مراجعة يومية منفصل عن المحادثة
• تتبع: سهل / صعب / نسيت لكل بطاقة
2. CEFR (A1 → C2)
• تحديد المستوى تلقائياً عند بدء المحادثة
• prompt النظام يتغير حسب المستوى
• تتبع: معدل الأخطاء، تنوع المفردات
• شريط تقدم مرئي A1→A2→B1→B2→C1→C2
3. Input +1 (كراشن)
• المعلم يستخدم 80% مفردات مألوفة + 20% جديدة
• يكتشف الكلمات الجديدة ويشرحها في السياق
• Claude prompt يتضمن قائمة المفردات المعروفة
• لا مفردات صعبة جداً ولا سهلة جداً
4. Gamification
• XP نقاط لكل رسالة صحيحة / مراجعة SRS
• Streak يومي مع تنبيه إذا انكسر
• شارات: "أول جلسة"، "7 أيام متواصلة"، إلخ
• مستويات: مبتدئ → متوسط → متقدم → خبير
