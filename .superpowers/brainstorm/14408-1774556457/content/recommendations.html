توصياتي الإضافية

اختر ما تريد إضافته للمشروع

🎙️

الصوت (TTS + STT) — الأهم

تشغيل صوت المعلم بنطق صحيح (TTS) + التحدث بالميكروفون للتدرب على النطق (STT). أساسي جداً لأي تطبيق لغات احترافي. يستخدم Web Speech API مجاناً في المتصفح.

📚

وضع الدروس المنظمة

إلى جانب المحادثة الحرة، درس يومي هيكلي: قاعدة نحوية + أمثلة + تمرين. مثلاً "اليوم: تصريف الأفعال المضارعة بالتركية". يكمّل SRS بدل ما يكون مجرد محادثة.

📱

PWA — تطبيق على الجوال

تحويل الموقع لـ Progressive Web App يتيح تثبيته على شاشة الجوال كتطبيق أصيل، وعمل مراجعات SRS بدون إنترنت. مهم لأن التعلم اليومي يحدث في الجوال غالباً.

📤

تصدير لـ Anki

تصدير بطاقات المفردات المجمّعة بصيغة .apkg لاستخدامها في Anki. يربط تطبيقك مع أقوى نظام SRS في العالم. بضع سطور كود فقط.

⚠️ ما لا أنصح بإضافته الآن
✗ نظام مستخدمين متعدد — الاستخدام الشخصي لا يحتاجه، يزيد التعقيد
✗ تمييز الكلام بالذكاء الاصطناعي (Whisper) — Web Speech API كافية وأسرع
✗ اختبارات CEFR رسمية — تقدير تلقائي من المحادثة أدق وأسهل
